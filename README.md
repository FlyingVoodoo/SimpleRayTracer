# SimpleRayTracer

Простой трассировщик лучей на C++, реализующий базовый рендеринг с жесткими тенями.

## Возможности

- Черно-белый рендеринг
- Рендеринг сферы и плоскости
- Жесткие тени от одного источника света
- Простое диффузное освещение
- Вывод в формат PPM

## Требования

- CMake 3.10 или выше
- Компилятор C++ с поддержкой C++17 (GCC, Clang, MSVC)

## Сборка

```bash
mkdir build
cd build
cmake ..
make
```

## Запуск

```bash
./raytracer
```

Результат будет сохранен в файл `output.ppm` в текущей директории.

## Структура проекта

```
SimpleRayTracer/
├── include/          # Заголовочные файлы
│   ├── Vec3.h       # 3D вектор математика
│   ├── Ray.h        # Класс луча
│   ├── Camera.h     # Камера
│   ├── Shape.h      # Базовый класс для объектов
│   ├── Sphere.h     # Сфера
│   ├── Plane.h      # Плоскость
│   ├── Light.h      # Источник света
│   └── Scene.h      # Сцена с объектами и светом
├── src/             # Исходные файлы
│   ├── main.cpp     # Основной файл с рендерингом
│   └── ...          # Реализации классов
└── CMakeLists.txt   # Файл сборки CMake
```
