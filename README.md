# SimpleRayTracer

Простой трассировщик лучей на C++, реализующий базовый рендеринг с жесткими тенями.

## Возможности

- Черно-белый рендеринг (grayscale)
- Рендеринг сферы и плоскости
- Жесткие тени от одного источника света
- Простое диффузное освещение
- Градиентный фон (небо)
- Вывод в формат PPM (800x600 пикселей)

### Текущая сцена

Программа рендерит:
- Одну сферу радиусом 1.0 в позиции (0, 1, 0)
- Плоскость в позиции y = 0 (земля)
- Источник света в позиции (5, 5, 5)
- Камера в позиции (0, 2, 5) смотрящая на (0, 0, 0)

## Требования

- CMake 3.10 или выше
- Компилятор C++ с поддержкой C++17 (GCC, Clang, MSVC)

## Сборка

```bash
mkdir build
cd build
cmake ..
make
```

## Запуск

```bash
./raytracer
```

Результат будет сохранен в файл `output.ppm` в текущей директории.

## Просмотр результата

Файл PPM можно открыть в большинстве программ для просмотра изображений или конвертировать в PNG:

```bash
# С помощью ImageMagick
convert output.ppm output.png

# С помощью GIMP, Photoshop, или других редакторов изображений
# Просто откройте output.ppm в программе
```

## Структура проекта

```
SimpleRayTracer/
├── include/          # Заголовочные файлы
│   ├── Vec3.h       # 3D вектор математика
│   ├── Ray.h        # Класс луча
│   ├── Camera.h     # Камера
│   ├── Shape.h      # Базовый класс для объектов
│   ├── Sphere.h     # Сфера
│   ├── Plane.h      # Плоскость
│   ├── Light.h      # Источник света
│   └── Scene.h      # Сцена с объектами и светом
├── src/             # Исходные файлы
│   ├── main.cpp     # Основной файл с рендерингом
│   └── ...          # Реализации классов
└── CMakeLists.txt   # Файл сборки CMake
```
